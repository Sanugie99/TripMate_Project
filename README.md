# TripMate_Project

## ✈️ 소개
사용자 맞춤 여행 일정 자동 생성 & 지도 기반 여행 계획 플랫폼
개발기간:(2025.06.30 ~ 2025.07.15)
배포기간:(2025.07.15 ~ )


## 🛠 기술 스택
- Front: React (tripmate/)
- Back: Spring Boot (trip/)
- DB: H2 (test) / MySQL (prod)

## 📂 폴더 구조
- trip/ : 백엔드(Spring)
- tripmate/ : 프론트엔드(React)

## 🔧 브랜치 전략
- `main`: 배포용
- `dev`: 개발 통합

# 역할 
- 김태연 : 프론트환경 구성, 백엔드환경전체 구성, 웹 배포
- 이상욱 : 백엔드 오류 수정, react 화면 구성
